# Definition of Done

Uma entrega só é considerada concluída quando atender, no mínimo, aos critérios abaixo:

- Código versionado na branch apropriada.
- Pull request aberto com template preenchido.
- Revisão técnica realizada.
- Build local e CI aprovados.
- Typecheck e lint sem falhas bloqueantes.
- Regras de negócio e autorização implementadas no backend quando aplicável.
- Documentação atualizada quando houver impacto arquitetural, funcional ou operacional.
- Fluxo testado no ambiente local da equipe.
- Sem credenciais expostas ou dados sensíveis hardcoded.
- Critérios de aceite revisados com o escopo da sprint.
