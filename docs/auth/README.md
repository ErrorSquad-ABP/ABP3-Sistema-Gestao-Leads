# Autenticação e Acesso

Este diretório consolida a documentação operacional do estado atual de autenticação, sessão e autorização do projeto.

## Arquivos

- `current-state.md`: fotografia fiel da implementação atual de auth, sessão, RBAC e fluxos públicos do frontend.

## Objetivo

Evitar que o conhecimento sobre login, bootstrap de sessão, papéis e limitações atuais fique espalhado entre:

- código;
- PRs;
- comentários de revisão;
- memória do time.
